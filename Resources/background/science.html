The distinguished points p1 and p2 are two points in the SK model's configuration space--i.e., two configurations--that are a specified distance apart. Because the space is homogeneous, the choice of p1 is arbitrary. For the same reason, p2 may be any point at the correct distance from p1.

Distance between two configurations is the number of spins that differ. A single spin-flip will change any configuration into another one that is at a distance 1—i.e., 1 step away—from the original.

Four numbers are sufficient to define the fundamental properties of the model:

  • N, the dimensionality of the SK model
  • k0, the distance between p1 and p2
  • α1, the depth of the potential well centered on p1
  • α2, the depth of the well centered on p2
The thermodynamic properties also depend on:
  • T, temperature (alternatively expressed as β ≡ 1/T)

In general, the configuration space of a system is the space of all possible configurations of that system. Each degree of freedom in the system is assigned its own dimension in configuration space. Each possible configuration is a distinct point in the space.

A single configuration in the SK model consists of N binary variables (aka "spins") whose possible values are conventionally written "+1" and "-1". Accordingly, there are 2^N distinct configurations possible to the system. Each spin is a degree of freedom, so the configuration space has N dimensions.
